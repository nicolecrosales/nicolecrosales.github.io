Visual Hierarchy

Ahumada Pharmacy

The principle of visual hierarchy is present throug color. The navigation icon has red color what makes it to highlight. It is the first thing you see because you need to find the product that you want to buy. The second thing you see is the search element, this does not have red color but has gray color, the magnifying glass highlights, and has a bigger size, because is the second place where you can look for the desired product. Other elements are smaller and light grey. Of this way you can see the hierarchy throug the different elements.

PARC: Proximity

Bunches

Here we have present the principle of proximity throug all page. First on the top in the navegation section, this elements are together in one line, then a link about covid information, and then information, and then the product. Each line has a determinate distance which shows less proximity because are different element, but in the same section or line the elements have more proximity because they are related items.

Each product is next to its name and price, there is proximity and the distance beetwen them is little. But there is a greater distance beetwen the products. The proximity principle allows to have the information very well organized.

PARC: Repitition

Levi's

The logo of this brand is made up for a red background and white text. This elemntns are repeated throug the website.

The red color is used in the lines for giving contrast, and the same combination of colors are used in the message icon which has a red background and with icon.

Of this way we can see how repetition principle gives unity to the page.
